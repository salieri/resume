# @faust/llm-utils
Shared LLM utilities for prompt templating and model execution.

* Handlebars templates
* OpenRouter prompts
* Codex execution

## Basic Usage Patterns
```bash
# build package
pnpm build

# run tests
pnpm test:ci

# lint
pnpm lint:fix

# build docs
pnpm docs
```

## Browser Compatibility
By default, this package outputs `es2023` compatible ESM code. It is not suitable for direct browser use.
